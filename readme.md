# MR-T

Cli tool that translates to english

Helpful if you work with people using other languages.


# Install

```bash
npm i -g mr-t
```

# Usage

1. type
```bash
>mt-t funktioniert ok
```
2. read output
```bash
>works ok
```
3. profit!
